# paleoscripts
A python module that contains a collection of postprocessing scripts

## Prerequisites

 - python
 - numpy
 - pytest
 - xarray
 - matplotlib
 - geocat-viz
 - cartopy
 
## How to install the package

In the top `paleoscripts` directory, type

```
pip install .
```

## How to test the package

In the top `paleoscripts` directory, type
```
pytest
```
